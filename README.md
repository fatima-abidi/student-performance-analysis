# Student Performance Analysis

This project analyzes student performance using Python and Streamlit.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Streamlit

## Features
- Filter students by gender
- Study time analysis
- Internet access impact
- Visualizations of grades

## How to Run
1. Install libraries:
   pip install pandas matplotlib streamlit

2. Run:
   streamlit run app.py

## Dataset
Student Performance Dataset
